-> msfconsole

2) Create a new workspace

-> workspace -a testlab

3) Change the workspace to the testlab workspace

-> workspace testlab

4) Scan all ports using the Nmap all ports scan:

-> db_nmap -p- 10.10.10.15

-> db_nmap -p22,80,111,139,443,32768 -sV -O 10.10.10.15
